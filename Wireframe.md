# Wireframe (Text)

## 1) 메인 페이지
```text
+---------------------------------------------------------------+
| Logo                                     [로그인] [회원가입] |
+---------------------------------------------------------------+
|                                                               |
|                    [ 카테고리 선택 영역 ]                      |
|         [일상회화]  [비즈니스]  [여행]  [식당]  [쇼핑]         |
|                                                               |
|                         [ 퀴즈 시작 ]                          |
|                                                               |
+---------------------------------------------------------------+
```

## 2) 퀴즈 풀이 페이지
```text
+---------------------------------------------------------------+
| Japanese Quiz                    진행률 3 / 10   [신고하기]   |
+---------------------------------------------------------------+
| 문제 문장 영역                                                     |
| "..."                                                           |
|                                                                 |
| ( ) 보기 1                                                      |
| ( ) 보기 2                                                      |
| ( ) 보기 3                                                      |
| ( ) 보기 4                                                      |
|                                                                 |
|                                  [제출]                         |
+---------------------------------------------------------------+
```

## 3) 종료 분기 모달 (비회원)
```text
+------------------------------+
| 퀴즈를 완료했어요!            |
| 결과 저장/통계를 보려면       |
| 로그인 또는 회원가입이 필요해요|
|                              |
| [로그인]   [회원가입]         |
+------------------------------+
```

## 4) 종료 분기 영역 (회원)
```text
+---------------------------------------------------------------+
| 퀴즈 완료!                                                     |
| 정답률: 80%                                                     |
|                                                                 |
| [다음 문제]   [다시 풀기]   [틀린 문제 보기]                    |
+---------------------------------------------------------------+
```

## 5) 마이페이지
```text
+---------------------------------------------------------------+
| 마이페이지                                                     |
+---------------------------------------------------------------+
| [회원 정보 수정]                                                |
| -------------------------------------------------------------- |
| 통계 요약                                                       |
| - 누적 풀이 수                                                  |
| - 정답률                                                        |
| - 최근 7일 학습                                                 |
| -------------------------------------------------------------- |
| [오답노트로 이동] [통계 상세 보기]                              |
+---------------------------------------------------------------+
```

## 6) 오답노트 페이지
```text
+---------------------------------------------------------------+
| 오답노트                                                       |
+---------------------------------------------------------------+
| 문제ID 1234   오답횟수 3   최근 오답일 2026-02-18              |
| 문제ID 1288   오답횟수 2   최근 오답일 2026-02-17              |
| ...                                                             |
|                                                                 |
| [재학습 시작]                                                   |
+---------------------------------------------------------------+
```

## 구현 우선순위
1. 메인 페이지
2. 퀴즈 풀이 페이지
3. 종료 분기(회원/비회원)
4. 결과 페이지
5. 마이페이지
6. 오답노트/통계
